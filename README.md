# **ᴍɪᴛ ʟɪᴄᴇɴꜱᴇ**

**ᴄᴏᴘʏʀɪɢʜᴛ (ᴄ) 𝟤𝟢𝟤𝟧 [ᴛᴇᴀᴍᴊᴀᴘᴀɴᴇꜱᴇ](https://github.com/TeamJapanese/)**
